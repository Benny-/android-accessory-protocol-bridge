The android accessory bridge provides additional protocols on top of android accessory protocol

Before you can use the additional functionality, you should create a accessory connection from the package {@link nl.ict.aapbridge.aap} and feed in into a {@link nl.ict.aapbridge.bridge.AccessoryBridge}

Dbus methods can be invoked using a {@link nl.ict.aapbridge.dbus.Dbus}. Dbus signals can be received using a {@link nl.ict.aapbridge.dbus.DbusSignals} object.
